# TopMark `config` Command Family

TopMark exposes a `config` command group to inspect and scaffold configuration:

- [`topmark config check`](config_check.md) — validate the *effective merged* configuration and report diagnostics.
- [`topmark config dump`](config_dump.md) — show the *effective merged* configuration.
- [`topmark config defaults`](config_defaults.md) — show the *built-in* default configuration.
- [`topmark config init`](config_init.md) — print a *starter* configuration file based on the
  annotated default template.
