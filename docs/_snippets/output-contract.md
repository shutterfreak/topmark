> [!NOTE]
>
> - `-v` / `--verbose` and `-q` / `--quiet` apply only to **TEXT output**.
> - Markdown output is **document-oriented** and ignores these flags.
> - JSON/NDJSON output is **machine-readable** and also ignores these flags.
